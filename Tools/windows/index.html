    OEMパーティション削除

    「ディスクの管理」では削除できないのでdiskpartで削除する

    1. list diskでディスクデバイス一覧を確認
    2. select disk *で処理対象ディスクを選択
    3. list partitionでパーティション一覧を確認
    4. select partition *で処理対象パーティションを選択
    5. delete partition overrideでパーティション削除

    録画時間の設定

    ゲームバーの録画のデフォルトは最長2時間。2時間以上録画するには設定変更が必要。
    「Windowsの設定」->「ゲーム」->「キャプチャ」で、「記録の最大長」を変更する。

    試してないけど最長4時間かな。

    動画の結合

    AviUtl使用

    1. 1つ目のファイルを「ファイル」「開く」で選択。
    2. 2つ目以降のファイルを順に「ファイル」「追加読み込み」
    3. 最後に「ファイル」「エクスポート」「MP4 Export」

    準備としてmp4を扱うプラグイン(MP4Plugin)をセットアップしておく。
